# Lumida Translations

Community-maintained translations for [Lumida](https://lumida.app), a free browser-based health dashboard for Google Health, Fitbit, and compatible devices.

## Available languages

Translation files are located in the [`messages`](./messages) directory.

English (`messages/en.json`) is the source locale.

## Contributing

Translation corrections and new languages are welcome.

To contribute:

1. Fork this repository.
2. Create a new branch.
3. Update or add a translation file.
4. Open a pull request describing your changes.

Please:

* Do not rename or remove translation keys.
* Preserve placeholders such as `{name}`, `{count}`, and `{date}`.
* Keep the JSON structure consistent with `messages/en.json`.
* Use natural translations rather than literal word-for-word translations.
* Do not add new application features or translation keys without prior discussion.

## About Lumida

Lumida provides a clear browser-based interface for exploring health and activity data from Google Health, Fitbit, and compatible devices.

Learn more at [lumida.app](https://lumida.app).
